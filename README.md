# RiskRadar 🚨  
*Predict. Prevent. Protect.*

## 📌 Overview  
Every decision involves risk — but most teams are blind to what’s coming.  
**RiskRadar** is a real-time risk monitoring platform that helps users **spot, assess, and act on potential threats** before they escalate.  

Built at **Sunhacks 2025**, RiskRadar combines a simple frontend dashboard with a robust backend API to provide insights in seconds.

---

## ✨ Features  
- 🌐 **User-friendly Dashboard** – Enter key info and instantly see risks.  
- 📊 **Risk Analysis Engine** – Backend API processes inputs into clear risk metrics.  
- 🔗 **Seamless Frontend/Backend Integration** – Built with React + Flask.  
- ⚡ **Fast Deployment** – Hosted on Render (backend) + Vercel/Netlify (frontend).  
- 🛠️ **Future-Ready** – Scalable to integrate ML, live data feeds, and push alerts.  

---

## 🛠️ Tech Stack  
- **Frontend**: React + Vite + TailwindCSS  
- **Backend**: Flask (Python), Flask-CORS, Requests, Dotenv  
- **Deployment**: Render (API), Github/Netlify, HTML/CSS (Frontend)  
- **Version Control**: GitHub
